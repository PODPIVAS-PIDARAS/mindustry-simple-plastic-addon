# mindustry-simple-plastic-addon
# Simple Plastic Addon

Adds a new item **Plastic** and a new block **Plastic Crafter** to Mindustry.

## Features

- New item: `plastic`
- New crafter block: `plastic-crafter`
  - Consumes coal and power
  - Outputs plastic

## Installation

### In-game (recommended)

1. Open **Mods** in Mindustry.
2. Click **Import → GitHub**.
3. Paste:

   https://github.com/PODPIVAS-PIDARAS/mindustry-simple-plastic-addon

4. Enable the mod and restart the game.

### Manual

1. Download this repository as ZIP.
2. Put the ZIP file into your `mods` folder.
3. Restart Mindustry.
